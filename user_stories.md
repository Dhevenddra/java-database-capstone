## Admin User Stories
- As an Admin, I want to be able to log into the administrative dashboard securely, so that I can access management features.
- As an Admin, I want to be able to add new Doctors to the clinic's roster with their specialties and available times, so that patients can start scheduling appointments with them.
- As an Admin, I want to be able to view a list of all doctors and their basic details, so that I can monitor staffing levels.
- As an Admin, I want to be able to search and filter doctors by name, specialty, or time, so that I can quickly locate specific medical staff.
- As an Admin, I want to be able to delete doctors who are no longer practicing at the clinic, to keep the directory updated and prevent invalid bookings.

## Patient User Stories
- As a Patient, I want to be able to log into my portal, so I can access my medical bookings and records.
- As a Patient, I want to search for available doctors by name or specialty, so I can find the right specialist for my health issue.
- As a Patient, I want to book an appointment with a doctor for a specific date and time, so I can ensure I am seen by a healthcare professional.
- As a Patient, I want to view all my upcoming and past appointments, so I can keep track of my clinic visits.
- As a Patient, I want to view my past prescriptions, so I can remember what medication was prescribed to me for previous conditions.

## Doctor User Stories
- As a Doctor, I want to log into my dedicated dashboard, so I can manage my patients and schedule.
- As a Doctor, I want to view a daily list of my booked appointments, so I can prepare for my consultations.
- As a Doctor, I want to review a patient’s past prescriptions and medical history during their visit, so I can provide accurate medical advice.
- As a Doctor, I want to issue a new digital prescription for a patient after a session, including medication names and dosages, so the pharmacy can process it.
- As a Doctor, I want to filter my appointments by specific dates, so I can schedule my personal commitments around my working hours.
